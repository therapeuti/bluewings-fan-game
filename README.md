# Bluewings Typing Practice

그랑블루를 위한 타자 연습 프로젝트입니다.
새로 산 키보드 테스트도 하고 응원가도 외우려고 만들었습니다.
현재는 `단어 연습`, `단문 연습`, `응원가 연습` 모드를 중심으로 동작합니다.
앞으로 게임같은 것도 추가할 계획입니다.

## 특징

- `단어 연습`: 수원 관련 단어를 빠르게 입력하는 모드
- `단문 연습`: 짧은 문장을 한 줄씩 입력하는 모드
- `응원가 연습`: 응원가 제목과 가사를 한 줄씩 순서대로 입력하는 모드

### 1. 의존성 설치

```bash
npm install
```

```bash

```

기본적으로 Vite 개발 서버가 실행됩니다.

### 3. 프로덕션 빌드

```bash
npm run build
```

### 4. 빌드 결과 미리보기

```bash
npm run preview
```

## 데이터 파일

### `word-list.md`

- 단어 연습에 사용되는 단어 목록
- 한 줄에 하나씩 작성
- 빈 줄 없이 관리하는 것을 권장

예시:

```text
수원
블루윙즈
프렌테트리콜로
```

### `sentence-list.md`

- 단문 연습에 사용되는 문장 목록
- 한 줄에 하나씩 작성

예시:

```text
- 수원 언제나 우린 너와 함께 해!
- 사랑한다 나의 사랑 나의 수원
```

### `song-list.md`

- 응원가 연습에 사용되는 응원가 데이터
- `#제목` 형식의 줄이 응원가 제목
- 그 아래 줄들이 해당 응원가 가사
- 가사 줄의 앞뒤 공백은 로딩 시 제거됨

예시:

```text
#데스파시토
너만이 우리의 가슴을 뛰게 해
이곳에 있어 프렌테 트리콜로
달리는 심장의 리듬을 함께 해
```

## 응원가 연습 동작 방식

- 사이드바 상단에서 응원가를 직접 선택할 수 있습니다.
- `랜덤 응원가`를 선택하거나 랜덤 버튼으로 다른 응원가를 바로 시작할 수 있습니다.
- 제목부터 가사까지 한 줄씩 순서대로 입력합니다.
- 현재 줄과 입력칸 위치는 고정되고, 지난 줄과 다음 줄만 이동합니다.
- 완료된 줄은 사용자가 실제로 입력한 문장을 기준으로 보여주며, 틀린 글자는 표시됩니다.
- 다음 줄 이동 규칙:
  - 입력 길이가 제시 문장보다 길어지면 다음 줄로 이동
  - 입력 길이가 제시 문장과 같을 때 `Space` 또는 `Enter`를 누르면 다음 줄로 이동

## 정확도 계산

- 단어/단문 연습은 입력 문자 기준 정확도를 사용합니다.
- 응원가 연습은 완료된 각 줄에서 발생한 오타 글자 수를 기준으로 정확도를 계산합니다.
